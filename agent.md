# Ellen Keyo School — Project Rules

## Brand
- Project: Ellen Keyo School website
- Language: Ukrainian
- Stack: HTML + CSS + JS (single index.html file)

## Colors
- Background: #eeede9
- Text: #1A1A1A
- Accent / Logo: #a32318
- Cards background: #ffffff
- Borders: #D5D0C8
- Footer background: #eeede9

## Typography
- Logo & Headings: Afacad Flux (Google Fonts)
- Body text: Inter (Google Fonts)
- Chillax Variable — use only if available locally

## Design Style
- Light, minimal, editorial — modern art gallery feel
- Mobile-first, fully responsive
- No gradients, no heavy shadows
- Rounded corners on cards: 16px
- Thin borders: 0.5px solid #D5D0C8

## Structure
1. Header — sticky, logo left (Ellen Keyo, #a32318), burger menu right
2. Hero — centered, Afacad Flux heading
3. Divider line (margin 0 24px)
4. Courses grid — 2 columns mobile, 4 desktop
5. Divider line
6. About section — label + photo left + text right
7. Footer — centered, beige background

## Courses
1. Блок 01 — Midjourney: основи та візуальне мислення
2. Блок 02 — Аватар та комерційні продукти  
3. Блок 03 — Анімація, відео та монтаж
4. Повний курс — AI Режисер (featured, border #a32318)

## DO NOT
- No dark footer backgrounds
- No all-caps logo (write: Ellen Keyo, not ELLEN KEYO)
- No extra animations or heavy effects
- No placeholder lorem ipsum text
